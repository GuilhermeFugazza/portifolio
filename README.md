# Portfolio profissional (Vite + React + Tailwind)

Projeto estatico de portfolio para desenvolvedor full stack, com paginas separadas e conteudo em pt-BR.

## Como rodar

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run preview
```

## Estrutura

- `src/components`: componentes reutilizaveis
- `src/pages`: paginas principais
- `src/layouts`: estrutura base com navbar e footer
- `src/data`: dados mockados dos projetos
- `src/styles`: estilos globais
